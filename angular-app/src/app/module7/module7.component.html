dystrybuanta

Dystrybuantę zmiennej losowej definiuje sie jako:

jest to funkcja rzeczywista, zawierająca wszystkie informacje o opisywanym rozkładzie.
Własności dystybuanty:

Funkcja ta musi być prawostronnie ciągła i niemalejąca. W języku angielskim określana jest jako CDF lub cumulative distribution function.

współczynnik zmienności

Współczynnik zmienności pozwala na porównanie zmienności cechy w róznych rozkładach, dzięki swojemu względnemu charakterowi.

gdzie:
σ - odchylenie standardowe z próby
μ - średnia arytmetyczna z próby (nie może być zerem)
Przykład:
Próba A: σ = {{ stddev1 | number : '1.2' }} μ = {{ mean1 | number : '1.2' }} -> zmienność: {{ variation1 | number : '1.2' }}
Próba B: σ = {{ stddev2 | number : '1.2' }} μ = {{ mean2 | number : '1.2' }} -> zmienność: {{ variation2 | number : '1.2' }}
Zmienność próby A jest większa od mniejsza od równa zmienności próby B
